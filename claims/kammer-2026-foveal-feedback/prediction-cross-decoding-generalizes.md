Cross-decoding generalization is the standard MVPA tool for testing representational-format sharing across conditions. A null result would not refute the feedback hypothesis itself — feedback could still exist with a different code — but would refute the specific claim that the feedback code is interchangeable with the bottom-up code in foveal V1.
