The shared-format hypothesis is interpretively important because it constrains the kind of feedback signal that could be reaching V1: a predictive code that lives in the same vector space as the bottom-up code is consistent with classic predictive-coding accounts (the prediction is "in the language" of the predicted signal). A non-shared format would still permit feedback but would suggest a different functional role (e.g., gain modulation, attention).
